## Overview

The **BASC Student Services Hub** is a student-focused web guide that organizes common campus transactions into clear **step-by-step processes**, including **requirements**, **where to go**, and **useful templates**.

It is designed to help students avoid confusion, reduce repeated questions, and complete requests faster—by knowing exactly what to prepare before going to an office.

---

## Features

- **Fast Search** for services and offices
- **Step-by-step processes** per transaction
- **Requirements checklist** per service
- **Office directory** (location, hours, contact)
- **Templates** (copy + print-ready request letters)
- **Mobile-friendly UI** with a clean menu drawer
- **Easy content updates** using a single `data.js` file

---

## Services Covered

The hub can include (and is designed for) common transactions such as:

- Transcript of Records (TOR)
- Shifting / Change of Program
- INC Processing
- Grade Concern / Grade Not Posted
- Portal / MIS Access (MIS Office)
- Transfer / Transferee Process
- Passbook (Request / Lost Passbook)
- Certificate Requests (Enrollment, Good Moral, etc.)
- Proposals (student/org/officer proposals and requirements)

> You can edit/add services anytime inside `data.js`.
